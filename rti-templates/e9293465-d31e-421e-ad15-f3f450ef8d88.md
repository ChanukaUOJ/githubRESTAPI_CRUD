> An open-source framework for distributed cloud synchronization.

## 1. Introduction
The **Aurora Project** aims to bridge the gap between edge computing and centralized data centers. This document serves as the primary technical specification for version `2.4.0-stable`.

---

## 2. Core Features
- **Real-time Sync**: Uses WebSockets for sub-100ms latency.
- **End-to-End Encryption**: AES-256-GCM by default.
- **Pluggable Storage**: 
    - Amazon S3
    - Google Cloud Storage
    - Local Filesystem
